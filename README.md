# Todo-List
Java version todo list web app that is connect to SQLite database.

## Technologies Used
* Java 
* JSP
* Servlets
* SQLite
* MVC Design
* Bootstrap
* Tomcat

## Functionalities
* User able to add item to list (to database)
* User able to edit an item
* User able to delete an item from database
